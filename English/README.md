## 🎯 What is AI Sky Eye?

AI Sky Eye is an **intelligent desktop automation system** designed for Windows. It enables your computer to "see" the screen, "understand" UI elements, and automatically perform various operations.

### Core Capabilities
- 👁️ **Visual Recognition** - Identify buttons, input fields, and text on screen using AI
- 🧠 **Intelligent Decision Making** - Auto-wait for loading, handle exceptions, retry failed operations
- ⚡ **Automated Execution** - One-stop solution for clicking, typing, screenshots, and window management
- 📱 **Remote Control** - Control your computer remotely via mobile browser

---

## 💡 Why Do You Need It?

### Pain Points of Traditional Automation Tools
- ❌ Requires precise coordinates, breaks with resolution changes
- ❌ Rigid operations, fails if page loading is slow
- ❌ Stops on first error, no auto-recovery
- ❌ Cannot identify UI elements, only "blind" execution

### AI Sky Eye's Solutions
- ✅ **Smart Positioning** - Find elements by name/type, resolution-independent
- ✅ **Smart Waiting** - Auto-detect when page finishes loading
- ✅ **Exception Recovery** - Auto-retry with exponential backoff
- ✅ **Visual Understanding** - OCR text recognition, image comparison for change detection
