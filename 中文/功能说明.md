# 功能说明

## 📊 功能总览（22个核心模块）

### 1️⃣ 基础框架模块

#### 安全控制系统 `safety_config.py`
| 特性 | 说明 |
|------|------|
| 三级保护 | strict / normal / off |
| 敏感区域 | 屏幕边缘、系统菜单保护 |
| 操作限速 | 防止操作过快 |
| 审计日志 | 记录所有操作 |

**核心 API:**
```python
from screen_controller.safety_config import SafetyConfig

config = SafetyConfig(level="normal")
```

---

#### 多后端截图系统 `enhanced_screenshot.py`
| 后端 | 特点 | 优先级 |
|------|------|--------|
| dxcam | GPU 加速，最高性能 | 1 |
| mss | 跨平台，中等性能 | 2 |
| pillow | 纯 Python，最兼容 | 3 |

**核心 API:**
```python
from screen_controller.enhanced_screenshot import capture_screen
img = capture_screen()
```

---

### 2️⃣ UI 自动化模块

#### UI 元素识别 `ui_element_finder.py`
| 功能 | 说明 |
|------|------|
| 名称查找 | 通过文字模糊匹配 |
| 类型查找 | Button, Edit, Menu 等 |
| 坐标获取 | 自动计算元素中心点 |
| 缓存优化 | 1秒内重复查询加速 |

**核心 API:**
```python
from screen_controller.ui_element_finder import find_element, find_button
elem = find_element("确定")
button = find_button("登录")
```

---

#### 智能控制器 `smart_controller_v2.py`
| 功能 | 说明 |
|------|------|
| ai_click | 通过名称点击，自动找坐标 |
| ai_type | 通过名称输入，自动找输入框 |
| ai_menu | 智能选择菜单项 |
| wait_for_element | 等待元素出现 |

**核心 API:**
```python
from screen_controller.smart_controller_v2 import ai_click, ai_type
ai_click("确定")
ai_type("用户名", "admin")
```

### 5️⃣ 高级功能模块
