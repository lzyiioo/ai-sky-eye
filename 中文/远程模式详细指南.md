# 远程模式详细指南

**通过手机/其他设备远程控制 Windows 电脑**

---

## 📱 远程模式概述

### 什么是远程模式？

远程模式允许你通过 HTTP API 从任何设备控制 Windows 电脑：
- 📱 手机浏览器
- 💻 其他电脑
- 🤖 自动化脚本
- ☁️ 云端服务

### 架构图

```
┌─────────────┐      HTTP       ┌─────────────┐
│   手机/平板   │  ←──────────→   │   Windows   │
│   浏览器    │   (同一WiFi)    │   AI天眼通   │
└─────────────┘                 └─────────────┘
                                       │
                                       │ API
                                       ↓
                               ┌─────────────┐
                               │  Windows    │
                               │   桌面      │
                               └─────────────┘
```

## 🚀 快速开始

### 步骤 1: 启动远程服务

```bash
# 方法 1: 使用启动脚本

python start_remote.py
```

```powershell
# Windows 命令行
ipconfig
```

```
手机浏览器输入:
http://192.168.1.100:8888/health
```

## 📱 使用示例

### 场景 1: 手机控制电脑关机

```bash
# 1. 打开开始菜单
curl -X POST http://192.168.1.100:8888/ai_click \
  -d '{"target": "开始"}'

# 2. 点击电源
curl -X POST http://192.168.1.100:8888/ai_click \
  -d '{"target": "电源"}'

# 3. 点击关机
curl -X POST http://192.168.1.100:8888/ai_click \
  -d '{"target": "关机"}'
```

---

### 场景 2: 远程截图查看

```python
import requests
import base64
from PIL import Image
from io import BytesIO

# 获取截图
response = requests.get("http://192.168.1.100:8888/screenshot")
data = response.json()

# 解码并显示
img_data = base64.b64decode(data["image"])
img = Image.open(BytesIO(img_data))
img.save("remote_screenshot.png")
img.show()
```

### 场景 3: 自动化脚本

```python
import requests
import time

BASE_URL = "http://192.168.1.100:8888"

def auto_login():
    """自动登录示例"""
    # 输入用户名
    requests.post(f"{BASE_URL}/ai_click", 
                  json={"target": "用户名"})
    requests.post(f"{BASE_URL}/type", 
                  json={"text": "admin"})
    
    # 输入密码
    requests.post(f"{BASE_URL}/ai_click", 
                  json={"target": "密码"})
    requests.post(f"{BASE_URL}/type", 
                  json={"text": "123456"})
    
    # 点击登录
    requests.post(f"{BASE_URL}/ai_click", 
                  json={"target": "登录"})
    
    # 等待并截图确认
    time.sleep(2)
    response = requests.get(f"{BASE_URL}/screenshot")
    return response.json()

if __name__ == "__main__":
    result = auto_login()
    print("登录完成，截图已保存")
```

## 🔒 安全配置

### 1. 访问控制

```python
# 在 remote_api.py 中添加认证
from functools import wraps
from flask import request

API_KEY = "your-secret-key"

def require_auth(f):
    @wraps(f)
    def decorated(*args, **kwargs):
        key = request.headers.get('X-API-Key')
        if key != API_KEY:
            return jsonify({"error": "Unauthorized"}), 401
        return f(*args, **kwargs)
    return decorated

@app.route('/ai_click', methods=['POST'])
@require_auth
def ai_click():
    ...
```

### 2. IP 白名单

```python
ALLOWED_IPS = ['192.168.1.100', '192.168.1.101']

@app.before_request
def check_ip():
    if request.remote_addr not in ALLOWED_IPS:
        return jsonify({"error": "IP not allowed"}), 403
```

### 3. HTTPS 支持

```python
# 使用 SSL/TLS
from flask import Flask
import ssl

context = ssl.SSLContext(ssl.PROTOCOL_TLS)
context.load_cert_chain('cert.pem', 'key.pem')

app.run(host='0.0.0.0', port=8888, ssl_context=context)
```

## 🌐 外网访问

### 方法一：路由器端口转发

```
1. 登录路由器管理界面
2. 找到"端口转发"或"虚拟服务器"
3. 添加规则：
   - 外部端口: 8888
   - 内部IP: 你的电脑IP
   - 内部端口: 8888
4. 保存并应用

外网访问: http://你的公网IP:8888
```

### 方法二：内网穿透 (frp)

```bash
# 安装 frp
# https://github.com/fatedier/frp

# frpc.ini (客户端配置)
[common]
server_addr = your-server.com
server_port = 7000
token = your-token

[ai-sky-eye]
type = tcp
local_ip = 127.0.0.1
local_port = 8888
remote_port = 8888
```

```python
from flask_socketio import SocketIO, emit

socketio = SocketIO(app, cors_allowed_origins="*")

@socketio.on('subscribe')
```
