# 安装说明

## 📋 系统要求

- **操作系统**: Windows 10 / Windows 11
- **Python**: 3.8 或更高版本
- **内存**: 4GB+ 推荐
- **权限**: 管理员权限（部分功能需要）

---

## 🚀 快速安装

### 步骤 1: 安装 Python

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.8+ 版本
3. **重要**: 安装时勾选 "Add Python to PATH"

### 步骤 2: 创建虚拟环境（推荐）

```bash
# 创建项目文件夹
mkdir AI天眼通
cd AI天眼通

# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate
```

### 步骤 3: 安装依赖

```bash
# 基础依赖（必需）
pip install Pillow numpy

# UI 自动化
pip install uiautomation

# 截图后端（选一个或多个）
pip install dxcam          # 推荐：GPU 加速
pip install mss            # 备选：跨平台

# OCR 识别（选一个或多个）
pip install paddleocr      # 推荐：中文识别
pip install easyocr        # 备选：多语言

# Windows 系统功能
pip install pywin32

# 高级功能
pip install schedule       # 定时任务
pip install win10toast     # 系统通知
pip install flask          # 远程 API
```

### 步骤 4: 验证安装

```bash
python -c "from screen_controller.smart_controller_v2 import ai_click; print('✅ 安装成功！')"
```

## 🔧 常见问题

### Q: 安装 uiautomation 失败？
```bash
# 以管理员身份运行 CMD，然后：
pip install uiautomation --user
```

### Q: dxcam 安装失败？
```bash
# dxcam 需要特定依赖
pip install comtypes
pip install dxcam
```

### Q: PaddleOCR 安装很慢？
```bash
# 使用国内镜像
pip install paddleocr -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 提示缺少 win32api？
```bash
pip install pywin32
# 然后执行
python Scripts\pywin32_postinstall.py -install
```
