## рҹ”§ ејҖеҸ‘жҢҮеҚ—

### ж·»еҠ ж–°еҠҹиғҪ

#### 1. еҲӣе»әжЁЎеқ—ж–Үд»¶
еңЁ `src/screen_controller/` зӣ®еҪ•дёӢеҲӣе»әж–°ж–Үд»¶пјҡ
```python
# my_feature.py
"""
My Feature - еҠҹиғҪжҸҸиҝ°
"""
import logging
from typing import Optional

logger = logging.getLogger(__name__)

class MyFeature:
    """еҠҹиғҪзұ»"""
    
    def __init__(self):
        pass
    
    def do_something(self, param: str) -> bool:
        """жү§иЎҢж“ҚдҪң"""
        try:
            # е®һзҺ°д»Јз Ғ
            return True
        except Exception as e:
            logger.error(f"Error: {e}")
            return False

# еҝ«жҚ·еҮҪж•°
_feature: Optional[MyFeature] = None

def get_feature() -> MyFeature:
    global _feature
    if _feature is None:
        _feature = MyFeature()
    return _feature
```

#### 2. еҜјеҮәеҲ° __init__.py
зј–иҫ‘ `src/screen_controller/__init__.py`пјҡ
```python
from .my_feature import MyFeature, get_feature
```

#### 3. ж·»еҠ жј”зӨәд»Јз Ғ
еҲӣе»ә `demo_my_feature.py`пјҡ
```python
def demo():
    from screen_controller.my_feature import get_feature
    feature = get_feature()
    result = feature.do_something("test")
    print(f"Result: {result}")

if __name__ == "__main__":
    demo()
```

## рҹ“қ д»Јз Ғи§„иҢғ

### е‘ҪеҗҚи§„иҢғ
- **жЁЎеқ—**: snake_case (e.g., `my_feature.py`)
- **зұ»**: PascalCase (e.g., `MyFeature`)
- **еҮҪж•°**: snake_case (e.g., `do_something`)
- **еёёйҮҸ**: UPPER_CASE (e.g., `MAX_RETRIES`)

### ж–ҮжЎЈеӯ—з¬ҰдёІ
```python
def my_function(param: str, optional_param: int = 0) -> bool:
    """
    еҮҪж•°жҸҸиҝ°
    
    Args:
        param: еҸӮж•°иҜҙжҳҺ
        optional_param: еҸҜйҖүеҸӮж•°иҜҙжҳҺ
        
    Returns:
        иҝ”еӣһеҖјиҜҙжҳҺ
        
    Raises:
        ValueError: ејӮеёёиҜҙжҳҺ
    """
```
